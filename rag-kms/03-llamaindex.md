## 核心定位

LlamaIndex 是一个专注于**RAG (Retrieval-Augmented Generation)** 和 **AI Agent** 开发的框架，帮助企业构建基于私有数据的智能应用。

---

## 核心功能

### 1. RAG 管道

- **数据连接**：从多种数据源（文件、API、数据库）加载数据
- **数据处理**：转换、分块、向量化
- **检索优化**：多种检索策略支持

### 2. AI Agent 能力

- **Tool Use**：Agent 可以调用外部工具
- **知识图谱集成**：支持知识图谱增强检索
- **多模型支持**：OpenAI、Claude、本地模型等

### 3. 企业级特性

| 特性 | 说明 |
|------|------|
| **数据安全** | 私有数据全程不外泄 |
| **可观测性** | 内置追踪和监控 |
| **评估工具** | RAG 和 Agent 性能评估 |
| **生产就绪** | 从原型到生产的完整路径 |

### 4. 检索增强

```
用户查询 → 检索 → 增强 → 生成
              ↓
         私有知识库
```

---

## 技术特点

- **模块化设计**：各组件可独立使用和替换
- **丰富的连接器**：100+ 数据源连接器
- **灵活的索引**：向量、关键词、混合等多种索引类型
- **评估框架**：内置 RAGAS 等评估工具

---

## 应用场景

- **企业知识库问答**：基于私有文档的智能问答
- **客户支持自动化**：产品文档驱动的客服 Agent
- **代码助手**：基于代码库的智能开发辅助
- **数据分析**：结构化数据之上的自然语言查询
