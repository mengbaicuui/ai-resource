## 底层技术

### 核心技术栈

| 层级 | 技术 |
|------|------|
| **模型支持** | 75+ LLM 提供商（Claude、GPT、Gemini、本地模型） |
| **接口形式** | 终端 / 桌面应用 / IDE 扩展 |
| **协议** | LSP (Language Server Protocol) |
| **模型优化** | Zen（OpenCode 自研模型优化方案） |

### 技术架构

```
用户终端 / 桌面 / IDE
         ↓
┌─────────────────────────────────────────────┐
│            OpenCode Agent                    │
├─────────────────────────────────────────────┤
│  多会话并行 (Multi-session)                  │
│  计划模式 (Plan) / 构建模式 (Build)           │
│  代码理解 (LSP) / 上下文管理                   │
├─────────────────────────────────────────────┤
│            Zen 优化模型层                      │
│   (精选 + 基准测试的编程专用模型)              │
├─────────────────────────────────────────────┤
│  Claude │ GPT │ Gemini │ Copilot │ 本地模型   │
└─────────────────────────────────────────────┘
```

## 核心技术特点

### 1. 多形态交付

- **终端界面**：基于终端的交互体验
- **桌面应用**：独立桌面客户端
- **IDE 扩展**：可集成到现有编辑器

### 2. LSP 深度集成

- **自动加载**：根据项目自动选择正确的 LSP
- **代码理解**：深度理解代码结构和语义
- **精准补全**：基于语言服务器的智能补全

### 3. 多会话并行 (Multi-session)

- 同一项目可启动多个 Agent 并行工作
- 分工处理不同任务
- 结果汇总协同

### 4. 隐私优先 (Privacy First)

- **不存储代码**：不保存任何代码或上下文数据
- **适合敏感环境**：可在隐私敏感环境运行
- **企业友好**：适合对代码安全有要求的团队

### 5. 模型灵活性

| 方式 | 说明 |
|------|------|
| GitHub Copilot | 直接登录使用 |
| ChatGPT Plus/Pro | 登录 OpenAI 账号 |
| Zen | OpenCode 优化的精选模型 |
| Models.dev | 75+ LLM 提供商 |
| 本地模型 | 支持本地部署 |

### 6. 项目初始化

```
/init → 分析项目结构 → 生成 AGENTS.md → 建立项目上下文
```

### 7. 分享与协作

- 生成会话链接
- 团队共享参考
- 调试协助

## 使用方式

### 安装

```bash
curl -fsSL https://opencode.ai/install | bash
```

### 配置 Provider

```bash
/opencode
/connect
# 选择 GitHub Copilot / ChatGPT / Zen 等
```

### 初始化项目

```bash
cd /path/to/project
opencode
/init
```

### 两种模式

| 模式 | 快捷键 | 说明 |
|------|--------|------|
| Plan | Tab | 计划模式，只生成方案，不执行 |
| Build | Tab | 构建模式，执行代码修改 |
