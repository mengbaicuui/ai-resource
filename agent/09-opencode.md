## 2. 核心机制

### A. LSP 深度集成

| 特性 | 说明 |
|------|------|
| **自动加载** | 根据项目选择正确 LSP |
| **代码理解** | 深度理解代码结构 |
| **精准补全** | 基于语言服务器 |

### B. Multi-Session (多会话)

| 特性 | 说明 |
|------|------|
| **并行工作** | 同一项目多个 Agent 同时 |
| **任务分工** | 分头处理不同模块 |
| **结果汇总** | 协同完成复杂任务 |

### C. Plan/Build 双模式

| 模式 | 说明 |
|------|------|
| **Plan** | 仅生成方案，不执行 |
| **Build** | 执行代码修改 |

## 5. PM 视角总结

### 核心价值

| 价值点 | 说明 |
|--------|------|
| **开源免费** | 120K Stars 认证，生态成熟 |
| **隐私优先** | 不存储代码，适合敏感环境 |
| **模型灵活** | 75+ LLM 提供商，随意切换 |
| **Copilot 集成** | 可直接使用现有 Copilot 账号 |

### 适用场景

- ✅ 开源爱好者
- ✅ 隐私敏感环境
- ✅ 需要模型灵活切换
- ✅ 多形态使用（CLI/桌面/IDE）

### 不适用场景

- ⚠️ 需要最强 AI 能力（Claude Opus）
- ⚠️ 企业级支持需求
- ⚠️ GUI 依赖者（桌面版较新）
