## 1. 底层核心技术

### A. IDE 深度集成

JetBrains AI 的核心差异化：**深度嵌入 JetBrains 全家桶**，理解项目的一切。

```
┌─────────────────────────────────────────────────────────────┐
│                  JetBrains AI 深度集成                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   ┌────────────────────────────────────────────────┐    │
│   │  项目配置 (pom.xml / build.gradle / ... )      │    │
│   │  运行配置 (Run Configurations)                  │    │
│   │  框架理解 (Spring / Django / ... )            │    │
│   │  语言服务器 (IntelliJ 引擎)                    │    │
│   └────────────────────────────────────────────────┘    │
│                          ↓                                  │
│   ┌────────────────────────────────────────────────┐    │
│   │  JetBrains AI 助手                              │    │
│   │  知道如何运行 │ 知道如何调试 │ 知道框架        │    │
│   └────────────────────────────────────────────────┘    │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 3. 与其他主流 Agent 工具的异同

### 横向对比

| 维度 | Claude Code | Cursor | JetBrains AI | Copilot |
|:---|:---|:---|:---|:---|
| **IDE 集成** | CLI | VS Code | **所有 JB IDE** | VS Code/JB |
| **框架理解** | ⚠️ | ⚠️ | **✅ 深度** | ⚠️ |
| **运行配置** | ❌ | ❌ | **✅** | ❌ |
| **语言覆盖** | 通用 | 通用 | **50+ 语言** | 通用 |
| **AI 能力** | 强 | 强 | **中** | 中 |
| **免费** | ✅ | ❌ | ❌ | ❌ |

---

### 关键异同点

#### JetBrains AI vs. Cursor

| 同 | 异 |
|---|---|
| 都提供 AI 编程辅助 | Cursor AI 能力更强；JetBrains AI **框架理解更深** |

#### JetBrains AI vs. Copilot

| 同 | 异 |
|---|---|
| 都支持 JetBrains IDE | JetBrains AI **原生集成**；Copilot 是插件 |

## 5. PM 视角总结

### 核心价值

| 价值点 | 说明 |
|--------|------|
| **框架深度理解** | 知道如何运行/调试你的项目 |
| **50+ 语言** | 覆盖所有 JetBrains 支持的语言 |
| **企业级** | 可靠性和支持有保障 |
| **多 IDE 统一** | 一个订阅覆盖所有 JetBrains 工具 |

### 适用场景

- ✅ JetBrains 重度用户
- ✅ 企业级 Java/Kotlin/Python 开发
- ✅ 需要框架特定优化
- ✅ 需要统一团队编码规范

### 不适用场景

- ⚠️ AI 能力相对基础（不如 Claude/Cursor）
- ⚠️ Agent 自主性有限
- ⚠️ 需要付费订阅
