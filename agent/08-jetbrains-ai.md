---

## 底层技术

### 核心技术栈

| 层级 | 技术 |
|------|------|
| **底层模型** | 多个模型组合（支持自定义） |
| **架构模式** | IDE 深度集成的单 Agent |
| **上下文** | 项目结构、运行配置、框架理解 |

### 技术架构

```
用户指令 → IDE 上下文 → AI 模型 → 代码建议
     ↓
完全理解项目配置 + 运行调试 + 框架优化
```

---

## 核心技术特点

### 1. IDE 深度集成

- 完全理解项目配置
- 知道如何运行和调试
- 框架特定优化

### 2. 智能补全

- 上下文感知的代码补全
- 文档生成
- 代码审查

### 3. 团队协作

- 团队知识共享
- 编码规范强制执行
