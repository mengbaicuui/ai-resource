## 核心定位

> *"From routine pull requests to your hardest problems, Codex reliably completes tasks end to end."*

Codex 是 OpenAI 推出的**AI 编程合作伙伴**，专注于完成端到端的工程任务，如构建功能、复杂重构、迁移等。

## 核心功能

### 1. Multi-Agent 工作流

- **内置 Worktrees**：并行处理多任务
- **云端环境**：Agents 在隔离环境中工作
- **跨项目协作**：数周工作量压缩到数天

### 2. Skills (技能系统)

- 代码理解
- 原型设计
- 文档生成
- 与团队标准对齐

### 3. Automations (自动化)

- Issue 分类和优先级处理
- 告警监控
- CI/CD 流水线
- 自动处理常规但重要的工作

### 4. PR Review

- 深度代码审查
- 发现其他工具漏掉的 bug
- 向后兼容性检查
- 提高团队代码质量基线

### 5. 多形态支持

| 形态 | 说明 |
|------|------|
| **Codex App** | Web 端入口 |
| **桌面客户端** | 独立桌面应用 |
| **CLI** | 终端命令行工具 |
| **IDE 集成** | 编辑器内直接使用 |
