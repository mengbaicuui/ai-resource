## 3. 与其他主流 Agent 工具的异同

### 横向对比

| 维度 | Claude Code | Cursor | Aider | Devin |
|:---|:---|:---|:---|:---|
| **交互形态** | Terminal | IDE | **Terminal** | Web |
| **开源** | ❌ | ❌ | **✅** | ❌ |
| **模型灵活性** | 仅 Claude | 多模型 | **极高（任意 LLM）** | 自研 |
| **Git 集成** | ✅ | ⚠️ | **✅ 原生** | ⚠️ |
| **入门门槛** | 高 | 中 | **中** | 高 |
| **免费** | ✅ | ❌ | **✅** | ❌ |

---

### 关键异同点

#### Aider vs. Claude Code

| 同 | 异 |
|---|---|
| 都是 CLI 工具 | Claude Code 仅限 Claude；Aider **支持任意 LLM** |

#### Aider vs. Cursor

| 同 | 异 |
|---|---|
| 都支持多文件修改 | Cursor GUI 体验更好；Aider **开源免费** |

## 5. PM 视角总结

### 核心价值

| 价值点 | 说明 |
|--------|------|
| **开源免费** | 完全免费，可自托管 |
| **模型灵活** | 支持任意 LLM，不被绑定 |
| **Git 原生** | 代码变更自动追踪 |
| **轻量快速** | 无 GUI，纯命令行 |

### 适用场景

- ✅ 开源爱好者
- ✅ 喜欢纯命令行
- ✅ 需要多模型对比
- ✅ 远程服务器开发

### 不适用场景

- ⚠️ GUI 依赖者
- ⚠️ 新手用户（CLI 门槛）
- ⚠️ 需要 IDE 深度集成
