# Aider

> 开源 CLI 编程工具

---

## 基本信息

| 项目 | 内容 |
|------|------|
| **类型** | 开源 CLI 工具 |
| **官网** | aider.chat |
| **GitHub** | github.com/paul-gauthier/aider |
| **支持模型** | GPT-4o、Claude 3.5、o1-preview 等 |

---

## 底层技术

### 核心技术栈

| 层级 | 技术 |
|------|------|
| **底层模型** | 任意支持 tool use 的 LLM |
| **架构模式** | 终端驱动的单 Agent |
| **版本控制** | 原生 Git 集成 |

### 技术架构

```
终端输入 → Aider (理解项目) → Tool Use → 代码修改
                ↓
           Git 自动提交
```

---

## 核心技术特点

### 1. 纯命令行体验

- 轻量级工具
- 适合远程服务器开发
- 可与任何编辑器配合

### 2. Git 深度集成

- **自动提交变更**
- **清晰的 diff 展示**
- **提交历史可追溯**

### 3. 多文件编辑

- 理解项目结构
- 跨文件修改协调
- 增量更新
