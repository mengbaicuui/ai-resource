## 底层技术

### 核心技术栈

| 层级 | 技术 |
|------|------|
| **底层模型** | 多个 LLM 组合（未公开具体） |
| **架构模式** | 云端沙箱执行 |
| **部署** | 一键部署到 Replit 托管 |

### 技术架构

```
需求 → AI 生成 → 云端运行 → 一键部署
  ↓
浏览器即 IDE，无需本地配置
```

---

## 核心技术特点

### 1. 云端开发环境

- **浏览器即 IDE**：无需安装任何东西
- **无需本地配置**：打开即用
- **即时环境复用**：每次打开相同环境

### 2. 一站式服务

- AI 生成代码
- 云端实时运行
- 一键部署

### 3. 实时协作

- 类似 Google Docs 的协作体验
- AI 和人类并肩编程
- 多人同时编辑
