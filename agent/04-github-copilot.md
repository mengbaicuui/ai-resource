---

## 底层技术

### 核心技术栈

| 层级 | 技术 |
|------|------|
| **底层模型** | OpenAI GPT-4 (定制版) |
| **架构模式** | Multi-Agent System (演进中) |
| **上下文** | 代码补全、注释生成、PR 摘要 |
| **企业安全** | Azure AD 集成、代码安全扫描 |

### 技术架构

```
需求理解 Agent → 代码生成 Agent → 测试 Agent → 审查 Agent
```

---

## 核心技术特点

### 1. 多 Agent 协作

- 多个专业 Agent 分工
- 代码编写、测试、审查协作
- 逐步向完全自主演进

### 2. 代码补全引擎

- **行级/函数级补全**
- **注释生成代码**
- **自然语言代码搜索**

### 3. GitHub 深度集成

- PR 描述自动生成
- Code Review 辅助
- Issue 自动分类

### 4. Workspace Agent (Preview)

- 自主创建分支
- 提交代码变更
- 初步的自主开发能力
