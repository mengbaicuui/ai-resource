## 核心定位

CrewAI 是一个专为**企业场景设计的多 Agent 框架**，使企业能够开发可以协作和自主运行的 AI 系统。

---

## 核心功能

### 1. 多 Agent 协作

- 多个 AI Agent 可以共同协作完成复杂任务
- Agent 之间可以通信和协调
- 支持任务分解和并行执行

### 2. 企业应用场景

| 场景 | 说明 |
|------|------|
| **物流** | 供应链优化、路径规划 |
| **客服** | 智能客服、多轮对话 |
| **制造** | 生产流程自动化 |
| **金融** | 风险评估、文档处理 |

### 3. 角色定义

- 为每个 Agent 定义明确角色
- 定义 Agent 的权限和能力
- 支持 Agent 之间的层级关系

### 4. 任务管理

- 任务分解和分配
- 任务状态跟踪
- 结果汇总和整合

```
    ↓

```
