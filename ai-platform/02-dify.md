## 核心定位

Dify 是一个**领先的 Agentic Workflow 构建平台**，专注于让团队能够轻松开发、部署和管理 AI 应用。

---

## 核心功能

### 1. 可视化工作流编排

- 拖拽式构建 AI 应用和工作流
- 支持多种任务类型
- 快速响应业务需求变化

### 2. RAG 管道

- 从多种数据源提取数据
- 转换并索引到向量数据库
- 优化 LLM 的数据获取能力

### 3. Multi-Agent 支持

- 多个 AI Agent 协作
- 复杂任务分解执行
- 灵活定义 Agent 角色和策略

### 4. MCP 原生集成

- 通过标准 MCP 协议访问外部 API、数据库和服务
- 消除集成复杂性和维护开销
- 支持 HTTP-based MCP 服务

### 5. 灵活的发布选项

- 一键发布为 MCP Server
- Dify 构建的工作流或 Agent 可作为标准 MCP 服务端
- 供无限多的 MCP 客户端使用
