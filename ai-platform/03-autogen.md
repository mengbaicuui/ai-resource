## 核心定位

AutoGen 是 Microsoft 推出的**开源多 Agent 对话编程框架**，使开发者能够构建基于多 Agent 协作的 LLM 应用。

---

## 核心功能

### 1. 多 Agent 协作

- 多个 Agent 之间可以进行对话协作
- Agent 可以扮演不同角色（用户、助手、审查员等）
- 支持 Agent 之间的层级结构和循环

### 2. 对话式编程

- 通过自然语言对话定义 Agent 行为
- 灵活的对话模式支持
- 代码生成和执行

### 3. 工具调用

- Agent 可以调用外部工具
- 支持函数调用和 API 集成
- 可扩展的工具注册机制

### 4. 多种工作模式

| 模式 | 说明 |
|------|------|
| **Agent Chat** | Agent 之间对话协作 |
| **Group Chat** | 群聊模式，多 Agent 同时参与 |
| **Sequential Chats** | 顺序对话，链式处理 |

---

## 技术特点

```
用户请求
     ↓
┌─────────┐     ┌─────────┐     ┌─────────┐
│ Agent 1 │ ←→  │ Agent 2 │ ←→  │ Agent 3 │
│ (助手)  │     │ (审查)  │     │ (执行)  │
└─────────┘     └─────────┘     └─────────┘
     ↓
   输出结果
```

- **可定制 Agent**：自定义 Agent 的角色和行为
- **人类参与**：支持人类在环（Human-in-the-loop）
- **代码执行**：内置代码生成和执行环境
